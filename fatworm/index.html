Introduction

Fatworm is the RDBMS design and implementation course of ACM Class, Shanghai Jiao Tong University. Students are required to work in teams to design and implement a complete database system from scratch, with support of a subset of SQL and JDBC. Query execution and optimization is encouraged as advanced topic. Student programs will be graded by an auto-grading system against various test-cases.

As my course project

I was in a team with my classmates Shijian Li and Liuli Chen for this course project, and wrote several components of the system as follows.

  1. Write a SQL parser in ANTLR v3
  2. Add JDBC support
  3. Design the interfaces between different phases
  4. Write the storage backend, including a buffer manager and an on-disk B+ tree implementation
  5. Write a low-level access layer like SQLJet, which allows one to manipulate and query records with keys and indexes directly (we employ this special design in order to work in parallel, since query plans can be naively translated to operations provided by this low-level access layer)
