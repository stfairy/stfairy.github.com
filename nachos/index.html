Background

Not Another Completely Heuristic Operating System, or Nachos, is instructional software for teaching undergraduate, and potentially graduate level operating systems courses. It was developed at the University of California, Berkeley, designed by Thomas Anderson, and is used by numerous schools.

Nachos in SJTU

This course project was adopted by ACM Class in Shanghai Jiao Tong University, and after some years of development, we had our own Nachos framework distribution in Java since 2009.

As my course project

I finished this course project in my third undergraduate year, and was required to build it from scratch and individually, including the major pieces of a modern operating system: (more details here)

  • Phase 1: Build a thread system
  • Phase 2: Multiprogramming
  • Phase 3: Caching and virtual memory
  • Phase 4: Networks and distributed system (discarded)
  • Phase 5: File system

At the end of the semester, I earned the full score in this course, and ranked 1 out of 31 students in our class.

Authors and contributors

This course project is required to be done individually, so I completed all the pieces of Nachos by myself.

The Nachos framework was originally adopted from this course and almost rewritten by teaching assistants from ACM Class.
