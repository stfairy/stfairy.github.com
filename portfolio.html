Portfolio of Xiao Jia

Bash

  • markpress is a simple blogging script written in Bash, designed to work with GitHub Pages. I wrote it for my own blog: blog.xiao-jia.com.

 

C/C++

  • Tradememo is a desktop software for futures transaction data analysis, written in C, PHP and JavaScript.

 

  • Single-sink wireless sensor network is a prototype system based on TinyOS and TelosB devices, written in C and Java. This is a teamwork with Menghui Wang and Shijian Li. (report)

 

Erlang & Haskell

  • Speculative nondeterminism is a research project in which we propose a novel concurrency control framework to allow real-time agents to speculate against the future by multiple exclusive choices that encode different strategies to achieve its goals. We developed a prototype system in Erlang and a simulator in Haskell.

Java

  • Tiger compiler is a course project written in Java, which targets the MIPS architecture, and leverages quadruples, linear scan register allocation, inter-procedural optimization, and garbage collection.

  • CoBrain is a distributed system for tasks like MapReduce, with a Web front-end for task management. This is a teamwork with Chunzhi Su, Hongyang Zhang, Li Han, and Linping Tang. (slides)

 

  • Commodities risk exposure and valuation system is a real-time system based on server-push technologies, leveraging ExtJS, EJB, and other J2EE technologies. The proposal of the requirements comes from Morgan Stanley. This is a teamwork with Jijia Wang and Liufeng Ren. (slides)

  • Nachos operating system is an operating system built from scratch, including the major pieces of a modern operating system: thread management, multiprogramming, virtual memory, file system, etc. (details)

  • Fatworm DBMS is a complete database system built from scratch, with support of a subset of SQL and JDBC. (details)

  • gogogo is a computer Go program developed for my artificial intelligence (AI) course. This is a teamwork with Shijian Li. (report)

JavaScript

  • dfj is an in-browser board game (like Minesweeper). This is a teamwork with Shijian Li. Play it now!

 

PHP

  • ACM class alumni is a personal profile showcase for alumni with social networking features. It is available to ~400 students from ACM Class.

 

  • TEP is a website developed for the Developer Weekend Hackathon by NUS Student Chapter of the ACM. It supports specialized, strongly-typed bookmarks for various websites such as GitHub, StackOverflow, CiteSeer, etc. There's also a Chrome extension for bookmarking.

 

  • SJTU map of stories is a website which allows anyone to create markers and story descriptions on a map of SJTU campus. (details)
