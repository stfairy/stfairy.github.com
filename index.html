My primary research area is programming language design & implementation.

My broader interests include program synthesis and software engineering.

I'm now a Junior Research Assistant of the Programming Languages and Systems Lab at NUS SoC, building a system in Erlang for speculative nondeterminism, which is a coordination construct that we proposed.

Course projects

(click to expand)

  1. Library management system
    • A system for library management, e.g. CRUD of books, users and privileges
    • Implemented a tiny Web server, based on Windows I/O completion ports
    • Designed an MVC application framework on top of my tiny Web server
    • Implemented a mini in-memory database with persistence support
    • Finished in three days
  2. Tiger compiler
    • Designed my own quadruples
    • Implemented the linear scan algorithm for register allocation
      (instead of the classical and boring graph coloring algorithm)
    • Fully optimized, especially for the famous eight queens puzzle (queens.tig)
      (actually I got the result of 166056 MIPS instructions executed in run-time, which is the best result for this course project in last 10 years, and maybe in the future :-)
    • Implemented a conservative garbage collector in MIPS assembly
  3. CoBrain (online demo)

    We built a distributed system for tasks like MapReduce, with a Web front-end for task management.

    This project is for the course Object-Oriented Analysis and Design.

    Click here to download my slides.

    This is a teamwork with Chunzhi Su, Hongyang Zhang, Li Han, and Linping Tang.

  4. Commodities risk exposure and valuation system

    We built a real-time system based on server-push technologies.

    This project is for two courses, Enterprise System Architecture, and Information System Analysis and Design. The proposal of the requirements comes from Morgan Stanley.

    Click here to download my slides.

    This is a teamwork with Jijia Wang and Liufeng Ren.

  5. Nachos operating system

    This project is to build an operating system from scratch, including the major pieces of a modern operating system: thread management, multiprogramming, virtual memory, file system, etc.

    See more details here.

  6. Wireless sensor network with a single sink node

    Click here to download our project report.

    This is a teamwork with Menghui Wang and Shijian Li.

  7. Fatworm DBMS

    This project is to build a complete database system from scratch, with support of a subset of JDBC interface.

    See more details here.

    This is a teamwork with Shijian Li.

  8. Computer Go

    Click here to download my project report.

    This is a teamwork with Shijian Li.
